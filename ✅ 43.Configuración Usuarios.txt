# 1. Para crear un usuario con la configuración predeterminada, simplemente usa:

# sudo useradd nombre_de_usuario
# sudo passwd nombre_de_usuario
# sudo useradd -m nuevo_usuario
# sudo useradd -m -d /opt/usuarios/otro_usuario otro_usuario
# sudo useradd -m -s /bin/bash usuario_con_bash


# 2.Comando Básico para Modificar un Usuario

# sudo usermod [opciones] nombre_de_usuario

# sudo usermod -s /bin/zsh usuario
# sudo usermod -s /sbin/nologin usuario_sin_login


# 3. Para eliminar un usuario sin eliminar su directorio personal ni sus archivos, utiliza:

# sudo userdel nombre_de_usuario
